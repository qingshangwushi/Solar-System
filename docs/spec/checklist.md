## 验证命令速查

```bash
# 静态
pnpm -r typecheck
cargo check --workspace
cargo clippy --workspace --all-targets -- -D warnings
cargo fmt --all -- --check
pnpm lint

# 单元
cargo test --workspace
pnpm -r test

# 构建
pnpm --filter web build
wasm-pack build crates/astro-core --target web

# 运行时
pnpm --filter web dev          # dev server
# 静态服务器托管 release 产物并断网冒烟
```
